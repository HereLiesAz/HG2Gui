# Contributing to HG2Gui

Thank you for your interest in contributing!

## Project Structure
This is a standard Android project.
*   **Source:** `app/src/main/java/com/hereliesaz/hg2gui/`
*   **Resources:** `app/src/main/res/`

## Coding Standards
*   **Language:** Java 8.
*   **Documentation:** All new code must be fully documented.
    *   **Class Javadoc:** Explain the purpose.
    *   **Method Javadoc:** Explain parameters and return values.
    *   **Inline Comments:** Explain *every* logical step. We aim for 100% comment coverage to ensure the code is understandable by anyone.

## Pull Requests
1.  Fork the repository.
2.  Create a feature branch.
3.  Make your changes.
4.  Ensure you have added "robust and exhaustive" documentation.
5.  Submit a PR.

## Building
Run `./gradlew assembleDebug` to build the APK.
